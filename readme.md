# BetterEvades

I like the game https://evades.io

I make evades.io for fun

Contribute if you want!

Frontend is in `public` folder

Backend is in `server` folder

That's about all.

# Running

`git clone https://github.com/TacoError/BetterEvades`

`cd BetterEvades`

`cd server`

`node .`
